This is a simple command line Tic Tac Toe game which can help you understand the principles of the minimax algorithm.
The Game can be played against another person or against the AI, you can also have the computer play against itself.
Choose 0, 1 or 2 players.
The bunndle includes a make file which can be used to produce the executables for your specific os. just cd into the directory where the source is located and type 'make'.
Feel free to use this source however you wish.
